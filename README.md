Terminal Dodge Game – README Improved Version • C Language • Console-Based Game This project is a feature-rich terminal game written in C. The player must dodge obstacles while collecting points. Improvements Included So Far:

Score System- Integer score counter that increases over time.
Player Lives System- Player gets 3 lives instead of instant game over.
Difficulty Levels- Speed increases as score increases.
Start Menu System- Includes Start, Instructions, Exit options.
Game Restart Option- Play Again without restarting the program.
High Score System (File Handling)- Stores highest score in highscore.txt.
Multiple Obstacles- Random spacing and multiple enemy objects.
Player Actions- Optional vertical movement included.
Improved ASCII Graphics- Better icons and lane layout.
Sound System (High Level)- Hooks added for future sound effects. Project Structure: /GameProject main.c highscore.txt README.md assets/ Compile: Windows: gcc main.c -o game Linux/Mac: gcc main.c -o game Controls: A/D - Left/Right W/S - Up/Down Q - Quit
